pub fn digit(i: Input<u8>) -> U8Result<u8>

Parses a single digit.

Note

The result is the code of the digit, no conversion takes place.

Example

use chomp::parse_only;
use chomp::ascii::digit;

assert_eq!(parse_only(digit, b"1"), Ok(b'1'));
